Everything you need to understand, evaluate, and implement Microsoft Fabric Planning. Built by practitioners, for practitioners.

## What is Fabric Planning?

Microsoft Fabric Planning is a new capability within the Microsoft Fabric platform that brings budgeting, forecasting, and financial planning directly into the unified analytics environment. For the first time, finance teams can build planning models alongside their existing Power BI reports and Fabric data pipelines.

This represents a potential shift away from standalone EPM tools and Excel-based planning toward an integrated, cloud-native approach. Fabric Planning is still evolving, and this knowledge hub exists to help finance and data professionals navigate what it can do today, where the gaps are, and how to get started.
